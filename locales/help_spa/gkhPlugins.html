Plugins

Hay algunos módulos plugin disponibles con la instalación por defecto:

  • Calculadora de expresiones, un plugin de servicio estándar.
  • Calendario, un plugin de servicio estándar.
  • Libro de referencia de nombres, un plugin de servicio estándar.
  • Cronología, un plugin de servicio estándar.
  • El plugin de entrada de datos en flujo agiliza la entrada masiva de datos desde las fuentes
  • El plugin de importación de pedigríes añade datos de un pedigrí a partir de texto externo.
  • El plugin de búsqueda de texto busca una entrada en la base de datos—funciona como un motor de búsqueda web.
  • El plugin del "juego de la vida de Conway" te ayuda a relajarte.
  • GEDmill, generador de sitios web.
