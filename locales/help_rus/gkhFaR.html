Найти и заменить

Эта функция выполняет поиск заданной строки и её замену на другое значение. В текущей реализации поиск выполняется только в персональных записях, только в именах, значениях фактов и ассоциациях.

Поддерживаются два типа дополнительных опций: учет регистра и подстановочные знаки. С включенной опцией учета регистра поиск будет выполняться только при точном соответствии регистра символов в искомой строке. С включенной опцией подстановочных знаков, при поиске будут учитываться маски фильтрации в искомой строке.

Внимание: перед использованием данной функции рекомендуется сделать резервную копию файла базы данных. При использовании массовой замены, рекомендуется сначала сделать замену в одном-двух случаях и только если всё соответствует ожидаемому - выполнять замену везде.
