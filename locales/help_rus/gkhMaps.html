Карты

Данный блок позволяет выводить на карты места рождения, смерти и местожительства из персональных данных, содержащихся в базе данных.

Для вывода карт используется ряд провайдеров: Google, Яндекс, OpenStreetMap, Bing, WikiMapia. Для смены провайдера используется кнопка "Провайдеры" на панели инструментов окна.

Карты можно сохранить в графический файл кнопкой на панели инструментов.

Для загрузки мест из записей базы данных используется кнопка "Загрузить места". Это основной режим, когда окно карт вызвано из меню "Родословная" главного окна приложения. Места загружаются согласно выбранным параметрам на левой панели. Может быть либо вывод выбранного типа места рождения, смерти или местожительства всех персон; либо вывод мест любых фактов отдельной выбранной персоны.

Если в настройках (меню Сервис\Настройки), на закладке "Интерфейс\Места и Карты", не включена опция "Поиск в геокодерах мест без координат", то при загрузке мест различных фактов, будет проверяться наличие места в справочнике записей локаций и места без координат будут пропущены. Если опция включена, то все места без координат будут проверяться по выбранному провайдеру геокодирования и названиям будут сопоставлены найденные варианты координат. Загрузка мест с поиском координат - медленный процесс. Рекомендуется использовать места, введенные в справочнике локаций, и с заданными координатами.

Следующие режимы предполагают, что используется ведение справочника локаций согласно стандарту GEDCOM 5.5 EL - т.е. локации образуют иерархию, в которой на каждом уровне (Город/Район/Область/Страна) заданы исторические имена локаций, с периодом действия этих имен, а также у каждой локации задан перечень локаций верхнего уровня, также с периодом действия подчинения.

  • При вызове карт из панели информации записей локаций - происходит выборка указанного места из фактов персон, места фильтруются для исключения дубликатов и исключения мест без координат. Доступны два подрежима:
    • "Карта мест" - у текущего места (Района, Области или Страны) выявляются все нижестоящие, подчиненные места и выводятся на карту.
    • "Карта персон" - у текущего места выявляются все подчиненные, затем выявляются люди, у которых указаны все обнаруженные места. Все отметки мест с указанием персон выводятся на карту.
  • При вызове карт из меню персоны в древе - происходит выборка всех мест из фактов персон древа, места фильтруются для исключения дубликатов и исключения мест без координат. После чего выводятся на карту с указанием перечня персон, которые были в местах на карте.

Персоны, обрабатываемые блоком карт, могут быть отфильтрованы общим механизмом фильтрации до того, как будет вызван блок карт.
